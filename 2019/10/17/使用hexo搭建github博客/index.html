使用hexo搭建github静态博客

2019/10/17 |
github
| hexo gitpages

创建 github repo

  1. 登录 github.com;
  2. 点击 New repository;
  3. 创建名称为 username.github.io的仓库，此处的username为github右上角显示的登录名，可以在Settings里修改;

安装 hexo

  1. 安装 nodejs

    // mac 上安装
$ brew install node
// Ubuntu 上安装
$ sudo apt-get install nodejs npm

  2. 查看是否安装成功

    $ node -v
$ npm -V

  3. 安装 hexo

    $ npm install hexo-cli -g
// 查看版本
$ hexo -v

创建写博客环境

    // 初始化目录
    $ hexo init blog
    $ cd blog
    // 安装所需插件
    $ npm install
    //////////////////////
    // 写新文章
    $ hexo now post "文章名称"
    // 启动 webserver
    $ hexo s

文章路径： blog/source/_post/文章名称.md

发布到github

在 _config.yml 文件末尾，有一个 deploy 配置

deploy:
    type: git
    repository: git@github.com:username/username.github.io
    branch: master

发布博客到github

$ hexo d -g

修改 hexo 主题

在 https://hexo.io/themes/ 中找到自己喜欢的主题，然后点击它的名字，就会跳转到对应的主题 repo （点击图片就会跳转到主题的预览），把这个 repo 克隆到 blog/themes/主题名称 (这里的主题名称最好和网站的一致)

例：

// 安装 Aath 主题
$ git clone -b master https://github.com/lewis-geek/hexo-theme-Aath.git themes/aath
// 安装主题依赖（一般在项目主页的说明中会说明需要安装哪些依赖）
$ npm install --save hexo-renderer-sass
// 然后修改 _config.yml 中的主题名称即可
// 至于更详细的配置，则要查看对应主题帮助文档 和 hexo 的帮助文档

Happy yourself！
